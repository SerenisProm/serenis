Avis de configuration de base de données (Firebase/Firestore)

Pour éliminer définitivement l'erreur "Missing or insufficient permissions" en mode production, rendez-vous dans la console Firebase et remplacez les **Règles de sécurité Firestore** par le modèle ouvert suivant : 

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
